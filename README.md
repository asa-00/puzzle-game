# 🧩 FractaZen – Logic Puzzle Game

FractaZen is a logic-based puzzle game with adaptive AI coaching. Deduce hidden patterns, level up, and receive feedback from your personalized coach.

## 🚀 Play the Game
👉 [Launch FractaZen on GitHub Pages](https://your-username.github.io/fractazen/)

## 🧪 Preview Mode
Use `preview.html` to test specific settings:
```js
window.previewLevel = 5;
window.previewStyle = "hype";
window.debugHints = true;
```

## 🛠 Tech Stack
- React + TypeScript
- SCSS Modules
- Web Speech API
- Supabase (Leaderboard)
- GitHub Actions (Deploy)

## 📦 Run Locally
```bash
git clone https://github.com/your-username/fractazen.git
cd fractazen
npm install
npm run dev
```

## 🌐 Deploy to GitHub Pages
This project auto-deploys from `main` via GitHub Actions.
Make sure to set `homepage` in `package.json`:
```json
"homepage": "https://your-username.github.io/fractazen"
```
